# Beauty Manager Application 


## Project Description

The back-end for the Beauty Manager mobile application – a platform that helps users manage visits to beauty and hair salons. This service provides all the necessary infrastructure to handle booking management, user authentication, notifications, and data storage.
The back-end is built with a serverless architecture on AWS to ensure high availability, scalability, and cost efficiency. It works in tandem with the front-end mobile app, exposing a set of secure APIs and integrating with cloud-native services.


## Technology Stack

### Language & Runtime

+ Node.js (TypeScript)
+ AWS Lambda

### Infrastructure

+ AWS CDK (Cloud Development Kit)
+ AWS CloudFormation

### Monitoring & Logging

+ AWS CloudWatch

### Database

+ Amazon DynamoDB (NoSQL)

### Other AWS Services

+ Amazon API Gateway
+ AWS IAM (permissions & roles)
